Offcanvas Sidebar

sidebar navigation menu 등으로 사용되면 modal 로 동작한다
.offcanvas: sidebar (컨테이너)생성
.offcanvas-start: offcanvas position 설정. 400px 폭으로 설정
.offcanvas-title: 적절한 margin 과 line-height 설정
.offcanvas-body: 이 안에 content 지정함
offcanvas 를 open 하려면 .offcanvas 의 id 를 지정하는 button 혹은 a 를 사용해야 한다
button 의 경우 data-bs-target="id" 를 사용하고
a 를 사용하는 경우 href="id" 를 사용해야 한다.

Right Offcanvas

.offcanvas-end 는 페이지의 오른쪽에서 offcanvas 를 등장하게 한다

Top Offcanvas

.offcanvas-top 는 페이지의 상단에서 offcanvas 를 등장하게 한다

Bottom Offcanvas

.offcanvas-bottom 는 페이지의 하단에서 offcanvas 를 등장하게 한다
