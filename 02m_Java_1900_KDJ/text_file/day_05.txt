조건문
    특정 조건에 따라 다른 코드를 실행시킬 때 사용하는 문법

if문
    if( 조건식 )
    {
        조건식이 참일 때 실행할 문장
        ... 
    }
    if문 바깥문장
    // if문 조건식이 참이라면 if문 안쪽의 문장을 수행하고
      만약 거짓이라면 안쪽의 문장을 실행하지 않는다
else문
    if ( 조건식 ) { 조건식이 참일 때 실행할 문장 }
    else { 조건식이 거짓일 때 실행할 문장 }

else if 문
    if ( 조건식1 ) { 조건식이 참일 때 실행할 문장 }
    else if ( 조건식2 ) { 조건식 1 거짓, 조건식2이 참일 때 실행할 문장 }
    else if ( 조건식3 ) { 조건식1, 조건식2 거거짓, 조건식3이 참일 때 실행할 문장 }
    else { 모든 조건식이 거짓일 때 실행할 문장 }

if vs 삼항연산자
    모든 삼항연산자는 if문으로 바꿀 수 있다
    모든 if문는 삼항연산자으로 바꿀 수 없다

    삼항연산자는 간단하게 나타낼 때 주로 사용한다.
    복잡하거나 가독성을 높이고자 할 때는 if문을 사용한다.
