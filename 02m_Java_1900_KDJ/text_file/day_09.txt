배열 (Array)
	여러 개의 값을 한 번에 저장하기 위해서 사용한다

배열의 선언과 초기화
	자료형[] 배열명 = {값, 값, 값, ... };
	ex) int[] arr = {80, 70, 55};

배열 속에 있는 개별 요소 사용하기
	인덱스 번호와 인덱스 연산자를 활용한다
	배열명[인덱스번호]
	

배열의 인덱스와 길이 간의 관계
	배열의 인덱스 번호는 배열의 길이 -1 이다

배열의 길이를 알려주는 키워드
	length
	배열명.length => 배열의 길이를 int값으로 받아온다

배열 선언
	자료형[] 배열명 = new 자료형[크기];
	ex) int[] arr = new int[10]

초기화 vs 선언만
	값을 나중에 대입할 경우에는 크기만 주어서 선언한 하는 것이 효율적
	미리 특정한 값을 넣어놓아야 한다면 초기화도 함께 해주는 것이 효율적

빠른 for문 / 향상된 for문 / foreach문
	for(자료형 변수명 : 배열명){}
	해당 변수에는 배열 속의 요소가 차례대로 담긴다
빠른 for vs 일반 for
	일반 for문 : 인덱스 번호가 필요하다면 (대입을 해야 한다면)
	빠른 for문 : 인덱스 번호가 필요 없이 배열 속에 있는 요소를 값으로 사용만 한다면
