클래스(class)
	사용자가 만드는 새로운 유형의 자료형(type)
	여러개의 값을 담아놓을 수 있다
	메소드도 넣어놓을 수 있다

클래스 선언(정의)
	class 클래스이름 {

	}

생성자
	클래스이름()
	메소드랑 똑같지만 이미 메모리에 객체를 등록시켜주는 기능이 정해져있기 때문에
	return 을 사용할 수 없고, 생성자 라고 부른다

	생성자를 정의한 것이 없다면 알아서 기본 생성자를 추가해 주지만
	만일 생성자를 직접 정의한 것이 있다면 기본 생성자를 추가해주지 않는다
	
	따라서 기본생성자와 초기화생성자를 모두 쓰고 싶다면 명시적으로 오버로딩 해야한다

초기화생성자
	클래스이름(값,값,...)
	해당 클래스 내의 변수에 값을 넣어주면서 동시에 메모리에 할당시켜주는 생성자
	객체화를 하면서 동시에 초기값도 설정하고 싶을 때 사용한다

this
	자기 자신 객체를 의미한다
	
static 제어자
	static 변수 : 모든 인스턴스가 값을 공유한다
			클래스명.변수명 으로 사용하는 것이 바람직하다
	static 메소드 : 모든 인스턴스에 동일하게 동작한다
			클래스명.메소드() 로 사용하는 것이 바람직하다

접근제어자 access modifier
			같은패키지	같은패키지	다른패키지
			같은클래스	다른클래스	다른클래스
	public		O	O	O
	protected	O	O	X(자식은가능)
	default		O	O	X
	private		O	X	X
