다차원 배열
	이차원 배열
	배열 속에 요소로 배열을 갖고 있다
	자료형[][] 배열명 = {{값, 값, ...}, {값, 값, ...}, ...};
	int[][] arr = {{}, {}, {}};

	이차원 배열의 선언
	자료형[][] 배열명 = new 자료형[크기][크기]
	int[][] arr = new 자료형[2][3]

메소드
	이름 뒤에 소괄호
	어떠한 기능을 한다
	코드를 재사용하기 위해서 사용한다
	
메소드 정의
	클래스 안쪽, 메인 메소드 바깥쪽에 정의한다
	public static void 메소드명 (){}

메소드의 사용(호출)
	클래스명.메소드명();

메소드는 매개변수가 있을 수도 있고, 없을 수도 있다
	만약 메소드를 만들 때 사용해야 하는 값을 메소드를 사용하는
	사람으로부터 받고 싶다면 매개변수로 설정할 수 있고,
	메소드를 사용하는 사람을 매개변수를 넘겨주어야지 사용할 수 있다

메소드는 return값이 있을 수도 있고, 없을 수도 있다
	만약 메소드를 사용한 결과를 값처럼 활용하고 싶다면
	return 값을 설정하여 값으로 되돌려주면 된다
	이 때 리턴타입 자리에는 리턴된 값의 타입을 써 주어야 한다
	메소드 사용자는 값으로 되돌려 받았기 때문에
	메소드 사용 결과를 변수에 저장하거나 추가적인 연산이 가능하다
